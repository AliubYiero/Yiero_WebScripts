# 更新日志

### 1.1.2 - update

- 在底部添加一个 `应用` 按钮, 用于不关闭对话框保存 CSS

### 1.1.1 - fix

- 修复了 `hightlight` 拼写错误，统一改为 `highlight`
- 修复代码高亮时光标位置丢失的问题，现在编辑时光标会保持在原位
- 修复 `LocalStorage.get()` 返回类型，现在正确返回 `T | null`
- 修复加载空 CSS 内容时的潜在问题
- 修复快捷添加元素时多余的换行符问题
- 新增 XSS 防护，自动清理 HTML 中的危险事件属性
- 重构代码：使用 `@yiero/gmlib` 替代内置的 `MenuManager`
- 提取选择器为常量，提高代码可维护性
- 更新 `hljs.highlightBlock` 为 `hljs.highlightElement` (API 迁移)

### 1.1.0 - feat

- 添加了一个菜单按钮, 用于快速在当前页面开启/关闭引入的CSS
- 现在可以使用 Tab 输入制表符了.

### 1.0.2 - fix

- 修复了在某些网页中, dialog的默认样式会被改变, 导致对话框默认会显示的问题

### 1.0.1 - style

- 给文本添加上字号样式, 防止被页面原本的 CSS 影响布局
- 现在输入框聚焦之后, 会有一个小的边框颜色改变提示.
- 给 "快速隐藏元素" 输入框添加上提示词.
- 优化了代码着色流程.

### 1.0.0 - feat

- 优化 UI, 现在可以统一编辑要写入页面的 CSS 样式了.
- 新增快捷输入框, 输入 css 选择器可快速将隐藏对应元素. 该功能对应原来的 "隐藏元素" 按钮.
- 代码高亮. 现在在输入完样式之后, 取消聚焦文本框就会对代码进行高亮着色.
