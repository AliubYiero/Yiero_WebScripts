# 微信读书自动阅读助手

## 概述

微信读书自动滚动、自动翻页脚本，解放双手，享受阅读。

## 功能特性

- 空格键控制：按空格键开启/关闭自动滚动
- 速度调节：按上箭头增加滚动速度，下箭头减小滚动速度
- 自动翻页：页面滚动到底部自动翻到下一页
- 底部留白：自动添加页面底部空白区域，方便阅读至页面末尾

## 配置项

| 配置名称 | 说明 | 默认值 |
|---------|------|--------|
| 滚动速度 | 页面自动滚动的速度 | 1 |
| 页面触底暂停时间 | 页面滚动到底部后暂停的时间（秒） | 3 |

## 使用方法

1. 安装浏览器扩展 [脚本猫](https://scriptcat.org/zh-CN)
2. 点击脚本详情页面的 **安装脚本** 按钮

## 快捷键说明

| 按键 | 功能 |
|-----|------|
| 空格 | 开启/关闭自动滚动 |
| 上箭头 | 增加滚动速度 |
| 下箭头 | 减小滚动速度 |
