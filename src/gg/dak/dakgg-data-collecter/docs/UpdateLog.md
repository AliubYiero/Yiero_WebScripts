### feat [0.4.0]

> > **更改数据来源, 变更特殊字符**. 
>
> - 更改大部分数据的来源: 直接读取响应数据, 减少从页面中获取的数据量, 使代码更加兼容. 
>   ***现在可以读取从 EA赛季 S1* ~ *正式赛季 Sn* 的任意内容了.** 
> - 更改了特殊符号的输出方式. 
>   现在特殊符号将被分为 **对局特殊符号** 和 **玩家特殊符号** 两种符号. 不再区分**主视角特殊符号**和**队友特殊符号**了, 因为**玩家特殊符号**支持输出所有玩家的信息. 
>   这项更改并不会影响之前的所有输出内容(除了部分输出文本修改). 
> - **删除**部分特殊字符: 
>   - 删除特殊字符: `%teammate%`. 
> - **修改**部分特殊字符: 
>   - 战术技能 `%skill%` 更改特殊字符为: `%tacticalSkill%` . 现在仍然可以通过 `%skill%` 输出战术技能, 但是该符号正在被废弃, 在未来版本可能被移除. 
>   - 战术技能等级 `%skillLevel%` 更改特殊字符为: `%tacticalSkillLevel%` . 现在仍然可以通过 `%skillLevel%` 输出战术技能等级, 但是该符号正在被废弃, 在未来版本可能被移除. 
>   - **修复部分特殊字符的文本输出**, 详细见下 *[更改数据内容-更改特殊字符]* .
> - **新增**特殊字符: 
>   - `%language%`: 显示玩家的注册地区. 
>   - `%skin%`: 显示玩家使用的皮肤
>   - `%versionMajor%`: 系统大版本号. 如 **v1.36.0** 中的 **1**.
>   - `%gameVersionMajor%`: 大版本号. 如 **v1.36.0** 中的 **36**.
>   - `%gameVersionMinor%`: 小版本号. 如 **v1.36.0** 中的 **0**.

**更改数据内容**

**新增特殊字符**

| 特殊字符               | 信息         | 新增内容                                                  |
| ---------------------- | ------------ | --------------------------------------------------------- |
| `%language%`           | 玩家注册地区 | `Chinese`: 中国<br />`Korean`: 韩国<br />`Japanese`: 日本 |
| `%skin%`               | 皮肤         | 玩家使用的皮肤                                            |
| `%versionMajor%`       | 系统大版本号 | eg. **v1.36.0** 中的 **1**                                |
| `%gameVersionMajor%`   | 大版本号     | eg. **v1.36.0** 中的 **36**                               |
| `%gameVersionMinor%`   | 小版本号     | eg. **v1.36.0** 中的 **0**                                |
| `%tacticalSkill%`      | 战术技能     | 同 `%skill%`                                              |
| `%tacticalSkillLevel%` | 战术技能等级 | 同 `skillLevel`                                           |

> 新增的 `%versionMajor%`, `%gameVersionMajor%` 和 `%gameVersionMinor%` 这三个特殊符号是因为输出方式更改, 导致无法直接获取版本号需要计算得来. 一般情况不需要使用, 还是和之前版本的一样使用 `%gameVersion%` 即可. 

**更改特殊字符**

| 特殊字符        | 信息     | 变更内容                                                     |
| --------------- | -------- | ------------------------------------------------------------ |
| `%rank%`        | 排名     | 直接显示数字排名, 而不是带有一个井号的 **#n** .<br />`#1` -> `1` |
| `%gameVersion%` | 版本号   | 如果小版本号为 0 , 将会直接显示<br />`v1.36` -> `v1.36.0`    |
| `%playTime%`    | 游玩时间 | 现在会精确到秒了, 即 **year-month-day hour:minute:second**<br />`2024-12-04 20:11` -> `2024-12-04 20:10:58` |
| `%time%`        | 游玩时刻 | 现在会精确到秒了, 即 **hour:minute:second**<br />`20:11` -> `20:10:58` |
| `%playerPage%`  | 玩家主页 | 玩家主页的链接将不再序列化, 而是直接显示玩家昵称. <br />`https://dak.gg/er/players/%EB%A1%9C%EB%A7%88%ED%8B%B1` -> `https://dak.gg/er/players/로마틱` |
| `%item%`        | 玩家出装 | 装备等级不再显示颜色, 而是显示稀有度. <br />`(紫)生命讯号感测器` -> `(英雄)生命讯号感测器` |

**对局特殊符号**

> **对局特殊符号** 并不包含玩家信息, 只是单纯的一些游戏对局信息. 

| 特殊字符        | 信息     |
| --------------- | -------- |
| `%game%`        | 游戏名   |
| `%season%`      | 赛季     |
| `%gameVersion%` | 版本号   |
| `%gameMode%`    | 游玩模式 |
| `%gameId%`      | 游戏ID   |

**玩家特殊符号**

> 所有**玩家特殊符号**可以输出任意一名玩家的信息. 直接使用为输出主视角信息, 添加前缀 `t1_` 为输出玩家一信息, 添加前缀 `t2_` 为输出玩家二信息. 
>
> 比如 `%player%` 输出主视角的玩家昵称, `%t1_player%` 输出玩家一的玩家昵称, `%t2_player%` 输出玩家二的玩家昵称. 其它的**玩家特殊符号**也同理. 
>
> > 除了上述的 5 个对局特殊符号, 之前版本的所有特殊符号都可以输出任意一名玩家的信息, 而不是仅仅能输出主视角的信息了, 虽然有些信息是重复的.

**删除特殊字符**

> 起初输出该特殊字符是因为 主视角 和 队友 的信息并不同步, 为了可能方便的参考所以输出了该字符串. 
>
> 现在主视角和队友能够输出的信息同步了, `%teammate%` 也就变成了一个彻底无用的符号了, 所以删去. 

| 特殊字符     | 信息               | 备注                                    |
| ------------ | ------------------ | --------------------------------------- |
| `%teammate%` | 队友信息(原始数据) | 仅供参考, 是一份几乎无法使用的文本数据. |
