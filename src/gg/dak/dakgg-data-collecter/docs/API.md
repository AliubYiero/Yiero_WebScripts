## 特殊字符

### 对局特殊符号

> **对局特殊符号** 并不包含玩家信息, 只是单纯的一些游戏对局信息. 

| 特殊字符             | 信息         | 内容                                                         |
| -------------------- | ------------ | ------------------------------------------------------------ |
| `%game%`             | 游戏名       | 常量: **永恒轮回**                                           |
| `%gameId%`           | 游戏ID       |                                                              |
| `%gameMode%`         | 游玩模式     | **一般** / **排位** / **钴协议**                             |
| `%season%`           | 赛季         | **正式赛季 Sn**<br />**季前赛 n**<br />**EA赛季 Sn**<br />**EA 季前赛 n** |
| `%rank%`             | 排名         |                                                              |
| `%avgRp%`            | 对局平均 RP  |                                                              |
| `%playTime%`         | 游玩时间     | **年-月-日 时:分:秒**                                        |
| `%date%`             | 游玩日期     | **年-月-日**                                                 |
| `%time%`             | 游玩时间     | **时:分:秒**                                                 |
| `%server%`           | 服务器       |                                                              |
| `%gameVersion%`      | 版本号       | eg. **v1.36.0**                                              |
| `%versionMajor%`     | 系统大版本号 | eg. **v1.36.0** 中的 **1**                                   |
| `%gameVersionMajor%` | 大版本号     | eg. **v1.36.0** 中的 **36**                                  |
| `%gameVersionMinor%` | 小版本号     | eg. **v1.36.0** 中的 **0**                                   |

### 玩家特殊符号

> 所有**玩家特殊符号**可以输出任意一名玩家的信息. 直接使用为输出主视角信息, 添加前缀 `t1_` 为输出玩家一信息, 添加前缀 `t2_` 为输出玩家二信息. 
>
> 比如 `%player%` 输出主视角的玩家昵称, `%t1_player%` 输出玩家一的玩家昵称, `%t2_player%` 输出玩家二的玩家昵称. 其它的**玩家特殊符号**也同理.  
>

| 特殊字符               | 信息                          | 备注                                                         |
| ---------------------- | ----------------------------- | ------------------------------------------------------------ |
| `%player%`             | 玩家名                        |                                                              |
| `%routeId%`            | 路径ID                        |                                                              |
| `%weapon%`             | 使用武器                      |                                                              |
| `%tacticalSkill%`      | 战术技能                      | 同 `%skill%`                                                 |
| `%tacticalSkillLevel%` | 战术技能等级                  | 同 `skillLevel`                                              |
| `%skill%`              | 战术技能                      | *deprecated*. 可能在后续版本中被删除, 请使用 `%tacticalSkill%` |
| `%skillLevel%`         | 战术技能等级                  | *deprecated*. 可能在后续版本中被删除, 请使用 `%tacticalSkillLevel%` |
| `%character%`          | 角色                          |                                                              |
| `%characterLevel%`     | 角色等级                      |                                                              |
| `%skin%`               | 皮肤                          | 玩家使用的角色皮肤                                           |
| `%rp%`                 | 初始 RP                       | eg. **8015**                                                 |
| `%changeRp%`           | 完赛增加 RP(带正负号的字符串) | eg. **+16** / **-18**                                        |
| `%addedRp%`            | 完赛增加 RP                   | eg. **16** / **-18**                                         |
| `%resultRp%`           | 完赛 RP                       | eg. **8031** / **7097**                                      |
| `%item%`               | 完赛出装                      | **武器 \|胸部 \|头部 \|手臂/装饰 \|腿部**                    |
| `%stat%`               | 击杀总数据                    | **TK / K / D / A**                                           |
| `%damage%`             | 造成伤害                      |                                                              |
| `%teamKill%`           | 团队击杀数                    |                                                              |
| `%kill%`               | 个人击杀数                    |                                                              |
| `%death%`              | 死亡数                        |                                                              |
| `%assist%`             | 助攻数                        |                                                              |
| `%damageFromPlayer%`   | 承受总伤害(幸存者)            |                                                              |
| `%traitCore%`          | 主潜能信息                    | eg. `主潜能: 抵抗型 |=> 金刚碎片 (抵抗型, 核心) -> 无惧感 (抵抗型, 辅助) -> 贪吃蛇 (抵抗型, 辅助)` |
| `%traitSub%`           | 副潜能信息                    | eg. `副潜能: 破坏型 |=> 狂乱 (破坏型, 辅助) -> 肉食主义 (破坏型, 辅助)` |
| `%skillOrder%`         | 技能加点顺序                  | `E -> Q -> W -> E -> E -> R -> E -> T -> D -> E -> T -> R -> D -> Q -> Q -> Q -> Q -> R -> D -> W -> W -> W -> W` |
| `%playerPage%`         | 玩家主页                      | https://dak.gg/er/players/xxxxx                              |
| `%language%`           | 玩家注册地区                  | `Chinese`: 中国<br />`Korean`: 韩国<br />`Japanese`: 日本    |
