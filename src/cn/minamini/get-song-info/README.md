# 获取歌单信息

## 概述

一个用于在 Minamini 网站爬取歌曲信息的油猴脚本。自动从页面表格中提取歌曲名称和歌手名称，支持批量采集并导出为 JSON 格式。

## 功能特性

- 爬取当前页面歌曲信息：从页面表格中提取歌曲名称和歌手名称
- 批量采集：自动去重，累积存储多页歌曲信息
- 一键导出：将采集的所有歌曲信息复制到剪切板（JSON 格式）
- 自动触发：点击页面内容区域时自动爬取当前页信息

## 使用方法

1. 安装浏览器扩展 [脚本猫](https://scriptcat.org/zh-CN)
2. 点击脚本详情页面的 **安装脚本** 按钮
3. 访问 [Minamini 网站](https://www.minamini.cn/)
4. 通过脚本菜单操作：
   - 点击「爬取本页信息」提取当前页歌曲
   - 点击「获取所有歌曲信息, 复制到剪切板」导出全部数据

## 输出示例

```json
[
	{
		"song": "歌曲名称",
		"singer": "歌手A & 歌手B"
	}
]
```
