# Bilibili直播隐藏重复弹幕

## 概述

在 Bilibili 直播间自动隐藏重复弹幕和刷屏用户的弹幕，净化直播聊天环境。

## 功能特性

- 隐藏重复评论弹幕：同一弹幕内容在评论区只显示一次
- 刷屏用户检测：n 分钟内发送 n 条弹幕的用户将被自动屏蔽
- 禁言用户过滤：被检测为刷屏的用户，其后续弹幕将被隐藏
- 直播弹幕去重：直播画面上的重复弹幕也会被隐藏

## 配置说明

脚本安装后可在脚本猫中配置以下参数：

| 参数 | 说明                       | 默认值   |
|------|--------------------------|-------|
| 刷屏检测间隔 | 检测刷屏的时间窗口（分钟）            | 2     |
| 刷屏次数 | 时间窗口内达到该次数则判定为刷屏，0 表示不检测 | 10    |
| 显示控制台日志 | 在控制台中显示屏蔽日志, 非开发者请勿开启    | false |

## 使用方法

1. 安装浏览器拓展 [脚本猫](https://scriptcat.org/zh-CN)
2. 点击脚本详情页面的 **安装脚本** 按钮
