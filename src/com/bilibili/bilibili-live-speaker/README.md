# Bilibili独轮车

## 概述

Bilibili独轮车是一个用于B站直播间的弹幕自动发送脚本。用户可以设置指定的弹幕内容、发送间隔和重复次数，脚本会按照配置自动在直播间发送弹幕。

> 本脚本与其它独轮车脚本的区别: 本脚本调用 Bilibili 的弹幕输入框发送弹幕, 所以理论上不怕封禁, 最多是有些弹幕会被发言频率检测发不出去.
> 
> 但还是请用户注意设置的发言频率. 

> 注意, 现在 Bilibili 使用动态发言频率检测, 最短的发言间隔是 500 ms, 但是发送过多弹幕会逐步提升检测频率, 会慢慢提升到 5000 ms. 

## 功能特性

- 自动发送弹幕：按照设定的时间间隔自动发送弹幕内容
- 随机间隔设置：支持设置最小和最大发送间隔，实际间隔在该范围内随机取值
- 重复次数设置：支持设置弹幕内容的最小和最大重复次数
- 菜单快捷控制：通过菜单按钮快速开启/关闭独轮车功能
- 配置保存：配置信息自动保存，下次使用时自动加载

## 使用方法

1. 安装网页拓展 [脚本猫](https://scriptcat.org/zh-CN)
2. 安装本脚本
3. 进入任意B站直播间（https://live.bilibili.com/*）
4. 点击脚本猫菜单中的"独轮车配置"设置弹幕内容和发送参数
5. 点击"开启独轮车"开始自动发送弹幕
6. 点击"关闭独轮车"停止发送

## 配置说明

### 弹幕内容
输入要发送的弹幕文本内容

### 发送间隔（毫秒）
- 最小间隔：两次发送之间的最短时间, 默认为 5000 ms, 最短为 500 ms
- 最大间隔：两次发送之间的最长时间
- 实际间隔会在最小值和最大值之间随机取值

### 重复次数
- 最小次数：弹幕内容的最少重复次数
- 最大次数：弹幕内容的最多重复次数
- 实际重复次数会在最小值和最大值之间随机取值
- 总长度超过直播间限制时会被截断
