# Bilibili视频倍速记忆

## 概述

一个用于 Bilibili 视频播放器的用户脚本，自动记忆视频播放倍速设置，并提供快捷键快速调整播放速度。支持跨视频保持倍速偏好。

## 功能特性

- **倍速记忆**: 自动保存用户设置的播放倍速，新视频自动应用上次使用的倍速
- **快捷键支持**: 通过键盘快捷键快速增加或减少播放倍速
  - 默认快捷键:
    - `Z` - 快速切换视频倍速，如果当前不是 1x 则重置到 1x，否则切换到上次记忆的倍速
    - `X` - 减少视频倍速
    - `C` - 增加视频倍速
- **实时显示**: 在视频播放器上显示当前倍速数值
- **跨视频同步**: 在不同页面内切换视频时保持倍速设置（默认关闭）
- **独立倍速UP主设置**: 为特定UP主设置独立倍速，不再参与全局同步，适用于听歌等场景
- **灵活配置**: 支持自定义倍速调整步长和快捷键组合

## 使用方法

1. 安装浏览器扩展 [脚本猫](https://scriptcat.org/zh-CN)
2. 点击脚本详情页面的 **安装脚本** 按钮
3. 打开任意 Bilibili 视频页面，脚本自动生效
4. 使用快捷键或播放器控制栏调整倍速

## 配置项说明

### 倍速配置

在脚本猫的管理面板中，可以修改以下配置项：

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| 倍速步长 | `number` | 0.25 | 每次增加或减少倍速的数值，建议范围 0.1 - 0.5 |
| 页面同步 | `boolean` | false | 切换视频时是否保持当前倍速设置 |

### 快捷键配置

支持自定义三组快捷键，通过下拉框选择按键或组合键：

**增加倍速快捷键**

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| 按键 | `select` | `C` | 触发增加倍速的按键或组合键（如 `C`、`Ctrl+C`、`Shift+X` 等） |

**减少倍速快捷键**

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| 按键 | `select` | `X` | 触发减少倍速的按键或组合键 |

**切换倍速快捷键**

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| 按键 | `select` | `Z` | 触发切换倍速的按键或组合键 |

### 独立倍速UP主设置

在视频页面的脚本菜单中，可以为当前UP主设置独立倍速：

1. 打开任意Bilibili视频页面
2. 点击浏览器工具栏上的脚本猫图标，或在页面右键菜单中找到脚本命令
3. 点击"设置独立倍速 (uid: xxx)"选项开启该UP主的独立倍速
4. 开启后，该UP主的视频将不再参与全局倍速同步，仅可通过快捷键调整
5. 如需恢复全局同步，点击"关闭独立倍速 (uid: xxx)"即可

**适用场景**：
- 听歌时需要固定倍速，而看视频时需要快进
- 特定UP主的内容需要单独设置倍速偏好
