# 更新日志

### 1.2.0 

- feat: 新增独立倍速UP主设置, 设置后对应UP主将不再全局同步倍速. 可以仅保留通过快捷键设置倍速的功能. 适用于听歌等场景. 
  - 添加 PlaybackRateSingle 类实现单页面独立倍速控制
  - 添加 renderSingleUpButton 渲染独立UP主倍速设置按钮
  - 添加 singleUpListStore 存储独立倍速UP主列表
  - 优化快捷键监听，使用 onKeydownMultiple 替代 onKeydown
  - 优化倍速切换逻辑，跳过 1.0 倍速中间值
  - 重构存储层，使用 createUserConfigStorage 和 GmArrayStorage
- feat: 修改用户配置的快捷键选择, 现在通过一个多选框选择. 优化脚本体积, 需要用户第一次加载完脚本之后才能够选择.
