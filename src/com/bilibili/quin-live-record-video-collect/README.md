# Mr.Quin 录播抓取程序 说明文档

> 随便写的, 不含任何筛选和排序逻辑, 仅爬取基础数据. 

## 概述

本脚本专为Bilibili平台设计，用于自动抓取Mr.Quin相关录播视频信息（包括直播日期、游戏名称、直播时长等关键数据），并支持生成结构化日志文件。脚本适配三位核心录播上传者（Mr.Quin本人、胧黑、自行车二层），可大幅提升录播资料整理效率。

## 功能特性

- **智能视频识别**  
  精准识别三位指定UP主（UID: 15810/245335/1400350754）上传的Mr.Quin录播视频，自动提取关键元数据
- **多格式日志导出**  
  支持一键生成Markdown表格, CSV格式的日志文件和JSON格式的原始数据，包含游戏名称、直播时间、视频链接等完整信息
- **持久化存储**  
  自动保存已抓取的视频记录，避免重复抓取，支持跨会话数据同步
- **分页自动处理**  
  智能遍历用户空间所有视频分页，确保无遗漏抓取
- **多重数据验证**  
  通过标题关键词、日期格式、BV号等多维度校验，确保抓取数据准确性
- **多游戏支持**  
  智能拆分单视频中的多游戏场次，确保每场游戏独立记录

## 使用方法

1. 安装网页拓展 [脚本猫](https://scriptcat.org/zh-CN)（推荐）或 Tampermonkey
2. 访问 [脚本安装页面](https://scriptcat.org/script-show-page/xxx) (注：实际安装链接需替换)
3. 点击 **安装脚本** 按钮完成安装
4. 访问以下任一页面激活功能：
    - [Mr.Quin个人空间](https://space.bilibili.com/15810/upload/video)
    - [胧黑的录播空间](https://space.bilibili.com/245335/upload/video)
    - [自行车二层空间](https://space.bilibili.com/1400350754/upload/video)

> **注意**：脚本仅在以上三个UP主的**视频上传页面**生效，其他页面不会显示操作菜单

## 示例

### 基础示例
1. 访问 [胧黑的视频上传页](https://space.bilibili.com/245335/upload/video)
2. 点击浏览器右上角脚本猫图标
3. 选择 **"抓取视频"** 菜单项
4. 控制台显示抓取进度（约1-2分钟，取决于视频数量）
5. 完成后选择 **"生成日志(md)"** 获取Markdown格式日志
```markdown
| 游戏名 | 直播时间 | 直播时长 | 上传者 | 标题 |
| ------ | -------- | -------- | ------ | ---- |
| 原子之心 | 2023年2月20日 | 06:25:51 | 胧黑 | [【Mr.Quin】2023年2月20日《原子之心》弹幕录像](https://www.bilibili.com/video/BV1hY4y1m7HV) |
```
