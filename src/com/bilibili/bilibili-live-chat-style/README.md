# Bilibili直播评论样式修改

一个用于自定义 Bilibili 直播间弹幕显示样式的用户脚本。

## 项目简介

本脚本修改 Bilibili 直播间的评论样式弹幕，使其按固定格式显示：用户信息在上方，弹幕内容在下方。支持自定义弹幕字号，并可在不刷新页面的情况下实时生效。

## 功能特性

- 修改直播间弹幕显示样式，用户信息在上，弹幕内容在下
- 支持自定义弹幕字体大小
- 配置项修改后实时生效，无需刷新页面

## 安装使用

### 前置要求

浏览器需要安装用户脚本管理器，推荐使用 [脚本猫](https://scriptcat.org/zh-CN)。

### 安装步骤

1. 安装脚本猫浏览器扩展
2. 访问脚本发布页面
3. 点击"安装脚本"按钮
4. 打开任意 Bilibili 直播间页面，脚本将自动运行

## 配置项

| 配置项 | 描述 | 类型 | 默认值 | 最小值 |
| ------ | ---- | ---- | ------ | ------ |
| 弹幕字体大小 | 修改弹幕显示的字号大小 | number | 16 | 12 |

配置修改后立即生效，无需刷新页面。
