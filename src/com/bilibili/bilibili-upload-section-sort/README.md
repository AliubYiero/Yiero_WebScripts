# Bilibili合集排序辅助 说明文档

## 概述

本脚本为Bilibili创作者中心提供合集视频排序功能，支持按**视频投稿发布时间**进行升序/降序排序，突破B站默认仅支持按标题排序的限制。在合集管理页面自动注入排序按钮，实现一键智能排序。

## 功能特性

-  **智能排序**：按视频真实投稿时间排序
-  **双模式支持**：
  - 升序排列（从旧到新）
  - 降序排列（从新到旧）
-  **本地缓存**：自动缓存视频发布时间，减少重复请求
-  **进度可视化**：显示数据加载倒计时，避免重复操作
-  **安全操作**：通过官方API执行排序，保障账号安全
-  **无缝集成**：自动识别合集管理界面，智能定位按钮位置

## 使用方法

1. 安装网页拓展 [脚本猫](https://scriptcat.org/zh-CN)
2. 访问脚本安装页面，点击 **安装脚本** 按钮
3. 进入B站创作者中心合集管理页面
4. 在合集视频列表上方将自动出现排序按钮 **[升序排序]** / **[降序排序]**
5. 点击任意一个排序按钮，排序完成后会在页面顶部显示排序成功消息
6. 刷新页面后可查看排序完成后的合集

> **注意**：首次使用需登录B站账号，脚本需要获取视频发布信息



**常见问题**  
❓ *为什么排序后需要手动刷新页面？*  
→ B站前端未设计实时更新机制，这是平台限制，非脚本问题

❓ *首次加载较慢是什么原因？*  
→ 需要获取每个视频的发布时间，视频发布时间需要通过B站API请求，频率过快的请求会导致账号异常。

❓ *按钮显示"加载数据中"但一直不完成？*  
→ 刷新页面，或请检查网络连接，或尝试清除脚本缓存（通过脚本猫管理界面）



## **API调用流程**

1. 监听B站合集数据请求
2. 提取合集内所有视频ID
3. 批量获取视频发布时间（带200ms间隔防封）
4. 根据选择的排序方式重组视频顺序
5. 通过官方API提交新排序
