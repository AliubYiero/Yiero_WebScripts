# Bilibili跳过视频广告

## 概述

通过 AI 智能分析 Bilibili 视频字幕内容，自动识别并跳过视频中的植入广告时间段，同时屏蔽评论区的广告跳转评论。

## 功能特性

- **AI 智能识别广告**: 利用 AI 分析视频字幕，自动检测广告时间段
- **自动跳过广告**: 播放到广告时段时自动跳转到正片内容
- **屏蔽评论广告**: 自动隐藏评论区中的商品推广链接评论
- **白名单/黑名单模式**: 支持按 UP 主配置分析策略
- **缓存机制**: 广告分析结果按 BV 号缓存，避免重复分析
- **可视化状态图标**: 在播放器上显示当前视频的广告识别状态

## 使用方法

### 安装

1. 安装浏览器扩展 [脚本猫](https://scriptcat.org/zh-CN)
2. 访问脚本发布页面，点击 **安装脚本** 按钮
3. 配置 AI 参数（首次使用）:
   - 点击脚本猫图标 -> 管理面板 -> 找到本脚本
   - 配置 API Key、API URL 和模型名称

### 配置说明

在脚本管理面板中可配置以下选项：

| 配置项 | 说明 | 示例 |
|--------|------|------|
| API Key | AI 服务提供商的 API Key | sk-xxxxxxxx |
| API URL | AI 接口地址 | https://api.openai.com/v1/chat/completions |
| 模型 | 使用的 AI 模型 | gpt-4o-mini |
| 屏蔽模式 | 分析策略：全部/白名单/黑名单 | 全部 |
| 白名单 | 仅分析指定 UP 主的视频 | UP主ID,用逗号分隔 |
| 黑名单 | 跳过指定 UP 主的视频 | UP主ID,用逗号分隔 |

## 技术原理

### 视频广告识别流程

1. **获取视频信息**: 提取 BV 号、UP 主信息
2. **检查缓存**: 查询是否已有该视频的广告分析结果
3. **获取字幕**: 调用 Bilibili API 获取视频字幕内容
4. **AI 分析**: 将字幕内容提交给 AI，识别广告时间段
5. **自动跳过**: 监听视频播放进度，到达广告时间点自动跳过

### 评论区广告屏蔽

- 遍历评论区 DOM（处理多层 Shadow DOM）
- 检测评论内容中的商品链接标识（`data-type="goods"`）
- 自动隐藏广告评论

## 注意事项

1. **AI 依赖**: 脚本需要配置有效的 AI API Key 才能分析广告
2. **字幕依赖**: 无字幕视频无法通过 AI 分析
3. **缓存管理**: 分析结果会缓存，如需重新分析可点击清除缓存按钮
