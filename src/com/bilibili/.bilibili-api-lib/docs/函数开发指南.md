## 函数开发步骤

### 步骤 1：定义接口类型

在对应模块的 `interfaces/` 目录下创建接口文件。

**命名规范**：`I{功能名称}.ts`，如 `IVideoInfo.ts`

```typescript
// src/video/interfaces/IVideoInfo.ts

/**
 * 视频信息接口
 */
export interface IVideoInfo {
  /** BV号 */
  bvid: string;
  /** 标题 */
  title: string;
  // ... 其他字段
}

/**
 * UP主信息
 */
export interface Owner {
  mid: number;
  name: string;
}
```

**要点**：
- 为每个字段添加 JSDoc 注释
- 复杂类型拆分为独立接口
- 导出所有需要使用的类型

---

### 步骤 2：实现 API 函数

在模块目录下创建 API 函数文件。

**命名规范**：`api_{操作}{资源}.ts`，如 `api_getVideoInfo.ts`

```typescript
// src/video/api_getVideoInfo.ts

import { xhrRequest } from '../xhrRequest.ts';
import type { IVideoInfo } from './interfaces/IVideoInfo.ts';

/**
 * 获取视频详细信息
 *
 * 详细描述这个 API 的功能。
 *
 * @param id - 参数说明
 * @param login - 是否需要登录
 * @returns 返回值说明
 *
 * @example
 * ```typescript
 * const video = await api_getVideoInfo('BV1xx411c7mD');
 * console.log(video.data.title);
 * ```
 *
 * @see [Bilibili API 文档链接](https://...)
 */
export function api_getVideoInfo(id: string | number, login: boolean = false) {
  // 参数处理
  const params: Record<string, string> = {};
  if (typeof id === 'string' && id.startsWith('BV')) {
    params.bvid = id;
  } else {
    params.aid = id.toString();
  }

  const url = 'https://api.bilibili.com/x/web-interface/view';
  
  // 根据是否需要登录选择请求方式
  if (login) {
    return xhrRequest.getWithCredentials<IVideoInfo>(url, { params });
  }
  return xhrRequest.get<IVideoInfo>(url, { params });
}

// 导出类型，方便用户直接使用
export type { IVideoInfo };
```

**要点**：
- 使用 JSDoc 编写完整的函数文档
- 包含功能描述、参数说明、示例代码
- 参考 Bilibili API 收集文档添加 `@see` 链接
- 导出相关类型

## 代码规范

### 命名规范

| 类型 | 命名规则 | 示例 |
|------|----------|------|
| API 函数 | `api_{操作}{资源}` | `api_getVideoInfo`, `api_editSeason` |
| 接口 | `I{名称}` | `IVideoInfo`, `ISeasonInfo` |
| 请求 Body | `I{操作}{资源}Body` | `IEditSeasonBody` |
| 工具函数 | 驼峰命名 | `getCsrf`, `formatTime` |

### 请求方法选择

| 场景 | 方法 |
|------|------|
| GET 请求，无需登录 | `xhrRequest.get<T>()` |
| GET 请求，需要登录 | `xhrRequest.getWithCredentials<T>()` |
| POST 请求，无需登录 | `xhrRequest.post<T>()` |
| POST 请求，需要登录 | `xhrRequest.postWithCredentials<T>()` |

### 参数处理

```typescript
// 1. 使用 params 对象传递查询参数
const params = { bvid: 'BV1xx411c7mD', page: '1' };
return xhrRequest.get<IData>(url, { params });

// 2. 使用 data 对象传递 POST body
const data = { title: '新标题', desc: '描述' };
return xhrRequest.postWithCredentials<IData>(url, { data });

// 3. 可选参数提供默认值
function api_example(param: string, optional: boolean = false) {}
```

## 完整示例

以新增 "获取用户关注列表" 功能为例：

### 1. 创建类型定义

```typescript
// src/user/interfaces/IUserFollowings.ts

export interface IUserFollowings {
  list: Following[];
  total: number;
}

export interface Following {
  mid: number;
  name: string;
  face: string;
}
```

### 2. 实现 API 函数

```typescript
// src/user/api_getUserFollowings.ts

import { xhrRequest } from '../xhrRequest.ts';
import type { IUserFollowings } from './interfaces/IUserFollowings.ts';

/**
 * 获取用户关注列表
 *
 * @param mid - 用户 mid
 * @param page - 页码，从 1 开始
 * @param pageSize - 每页数量
 * @returns 关注列表
 *
 * @example
 * ```typescript
 * const list = await api_getUserFollowings(123456, 1, 20);
 * console.log(list.data.list);
 * ```
 */
export function api_getUserFollowings(
  mid: number,
  page: number = 1,
  pageSize: number = 20
) {
  const url = 'https://api.bilibili.com/x/relation/followings';
  const params = {
    vmid: mid.toString(),
    pn: page.toString(),
    ps: pageSize.toString(),
  };

  return xhrRequest.getWithCredentials<IUserFollowings>(url, { params });
}

export type { IUserFollowings };
```
