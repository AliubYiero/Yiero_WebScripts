# 更新日志

### 1.0.0 - feat

- 原始样式修改为使用浮动布局.
- 修复表情弹幕填不满一行的问题.
- 添加和 [Bilibili直播评论样式修改](https://scriptcat.org/zh-CN/script-show-page/5757) 的联动样式

### 0.2.4 - fix

- 修复刚进入直播间时, 所有弹幕的时间都是刚进入直播间时的时间的问题.

### 0.2.3 - fix

- 修复自己发送的弹幕时间错乱的问题.

### 0.2.2 - pref

- 表情弹幕可以显示表情发送时间了
- 更改了弹幕时间文本的显示样式, 现在分为3种情况:
    - 表情弹幕: 跟在表情后面
    - 舰长弹幕: 弹幕文本单独分为1行, 弹幕时间文本在舰长框的最右边 (因为舰长背景颜色框不是满宽的, 所以只能在舰长框的右边)
    - 普通弹幕: 弹幕文本单独分为1行, 弹幕时间文本在聊天框的最右边

### 0.2.1 - pref

- 鼠标浮动到弹幕上时, 弹幕发送时的直播时长文本会高亮

### 0.2.0 - feat

- 鼠标浮动到弹幕上时, 切换显示弹幕发送时的直播时长

### 0.1.0 - feat

- 显示弹幕发送时的现实时间
