# Bilibili自动添加收藏

## 概述

本脚本旨在提升用户在Bilibili平台上的收藏效率。当您浏览视频页面时，脚本会自动将当前视频添加到指定名称的收藏夹中，并支持智能管理多个编号收藏夹，实现自动化、有序化的视频收藏整理。

## 功能特性

- **自动收藏**：进入任意视频播放页面后，脚本将自动执行收藏操作，无需手动点击。
- **智能收藏夹管理**：根据预设的收藏夹标题（如 `fun`），自动创建并使用按序编号的收藏夹（如 `fun1`, `fun2`...）；若当前最新编号收藏夹已满（上限1000个视频），则自动创建下一个编号的新收藏夹。
- **灵活配置**：通过菜单按钮可自定义收藏夹主标题。

## 使用方法

安装网页拓展 [脚本猫](https://scriptcat.org/zh-CN)，点击脚本详情页面的 **安装脚本** 按钮。

安装后，请通过油猴脚本管理面板进入本脚本的设置界面，完成以下配置：

1. **设置收藏夹标题**：
	1. **默认收藏夹**: 设置为 `默认收藏夹` 时, 会将视频收藏到无上限的默认收藏夹. 
	2. **自定义收藏夹标题**: 设定用于自动创建和匹配收藏夹的基础名称（默认为 `fun`）。例如设置为 `fun`，则脚本将使用 `fun1`、`fun2` 等进行收藏。


配置完成后，每次访问新的视频页面，脚本都会按照规则自动将其收藏至合适的收藏夹中。

## 示例

### 基础示例

假设您将“收藏夹标题”设置为 `fun`，且尚未创建任何相关收藏夹：
- 脚本检测到不存在 `fun1`，于是创建 `fun1` 并将当前视频加入其中。
- 后续继续观看其他视频，只要 `fun1` 未满1000个视频，所有视频都将被添加进 `fun1`。

### 高级示例

当前已有 `fun1` 至 `fun5` 收藏夹，其中 `fun5` 已满：
- 脚本识别出最新的非满收藏夹是 `fun5`，但因其已满，故自动创建 `fun6`。
- 当前视频被添加进新创建的 `fun6` 中。
- 随后，`fun5` 将被自动排序至收藏夹列表末尾，保持收藏结构清晰有序。

此机制确保您始终拥有一个可用的“最新”收藏夹，避免因单个收藏夹容量限制而中断自动收藏流程。
