# Bilibili直播自动点赞 说明文档

## 概述

Bilibili进入直播间后自动点赞. 本脚本会在用户进入直播间后自动进行点赞操作, 支持自定义延迟时间、关注过滤、数量限制等功能.

## 功能特性

- **自动点赞**: 进入直播间后自动执行点赞操作
- **随机延迟**: 可自定义点赞间隔的随机延迟范围, 避免频繁操作
- **关注过滤**: 可选择仅点赞已关注的主播
- **数量限制**: 可设置每个主播每天的最大点赞数量
- **动画控制**: 可屏蔽点赞动画, 减少界面干扰
- **计数显示**: 在脚本菜单中显示当前房间点赞数和今日总点赞数

## 配置项

| 配置项 | 类型 | 默认值   | 说明 |
|--------|------|-------|------|
| 点赞随机时间延迟 (ms) - 最小随机时间 | 数字 | 2000  | 点赞间隔的最小延迟时间 (无法少于500ms) |
| 点赞随机时间延迟 (ms) - 最大随机时间 | 数字 | 3000  | 点赞间隔的最大延迟时间 |
| 仅点赞关注用户 | 布尔 | true  | 开启后只点赞已关注的主播 |
| 每个主播每天点赞的最大数量 | 数字 | 330   | 每个主播每天的最大点赞数量 |
| 是否显示点赞动画 | 布尔 | false | 显示点赞后的动画效果 |
| 是否显示点赞数量 | 布尔 | true  | 显示点赞数量 |

## 使用方法

安装网页拓展 [脚本猫](https://scriptcat.org/zh-CN), 点击脚本详情页面的 **安装脚本** 按钮.

## 示例

### 基础示例

默认配置下, 脚本会:
1. 进入直播间后自动开始点赞
2. 每次点赞间隔随机 500ms ~ 5000ms
3. 仅点赞已关注的主播
4. 每个主播每天最多点赞 330 次

### 高级示例

调整配置项可以实现:
- **快速点赞**: 将最小延迟设为 500ms, 最大延迟设为 1000ms
- **仅点赞特定用户**: 开启"仅点赞关注用户"
- **限制用量**: 降低"每个主播每天点赞的最大数量"数值
