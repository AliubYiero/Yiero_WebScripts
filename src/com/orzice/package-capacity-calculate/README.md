# 三角洲数据帝自动计算背包容器性价比 说明文档

## 概述

本脚本专为《三角洲行动》玩家设计，在三角洲数据帝网站上自动计算背包容器的性价比。脚本通过添加"背包容量"列，允许用户自定义装备容量数据，并提供多种导出功能，帮助玩家快速分析不同背包与胸挂组合的性价比。

## 功能特性

- **容量数据管理**：在装备表格中添加"背包容量"列，支持实时编辑和保存用户自定义的容器容量
- **智能组合分析**：自动计算背包与胸挂的所有可能组合，包括单一装备和组合装备的性价比
- **多格式数据导出**：
  - 导出完整背包容器组合价格列表（CSV/表格格式）
  - 导出最佳性价比组合方案（CSV/表格格式）
- **本地存储支持**：自动保存用户输入的容量数据，下次访问时自动加载
- **响应式界面**：适配三角洲数据帝网站布局，不影响原站功能

## 使用方法

1. 安装网页拓展 [脚本猫](https://scriptcat.org/zh-CN)
2. 点击脚本详情页面的 **安装脚本** 按钮
3. 访问 [三角洲数据帝装备页面](https://orzice.com/v/zhanbei?n=背包) 或 [胸挂页面](https://orzice.com/v/zhanbei?n=胸挂)
4. 在表格中输入或修改各装备的容量值
5. 通过油猴脚本菜单（工具栏脚本猫图标）选择需要的数据导出选项

## 示例

### 基础示例

1. 访问 [背包页面](https://orzice.com/v/zhanbei?n=背包)
2. 在新增的"背包容量"列中输入各背包的实际容量
3. 点击脚本猫图标 > 选择"下载背包最佳性价比(csv)"
4. 系统自动生成并下载CSV文件，包含按性价比排序的背包方案

### 高级示例

**组合性价比分析：**
1. 同时访问[背包页面](https://orzice.com/v/zhanbei?n=背包)和[胸挂页面](https://orzice.com/v/zhanbei?n=胸挂)
2. 分别输入两类装备的容量数据
3. 使用"下载背包容器组合价格列表(table)"功能
4. 在Excel中打开下载的文件，使用筛选功能：
   - 按"总容量"筛选特定容量需求
   - 按"单格性价比"排序找出最优组合
   - 对比不同组合的"总价格"和"总装备"价值
