# 更新日志

## 0.4.1 - fix

- 修复快捷键错乱, 不生效的问题

## 0.4.0 - update

- 修改快捷键: 
  - 切换滚动状态: 按 `Space` 开启/关闭滚动
  - `PageUp` 和 `PageDown` 不再作为滚动状态切换快捷键使用, 回归页面滚动功能

## 0.3.0 - feat

- 优化滚动性能

## 0.2.0 - feat

- 增加速度：按 `Shift + PageUp` 键增加滚动速度
- 降低速度：按 `Shift + PageDown` 键降低滚动速度
- 智能暂停：切换标签页时自动暂停滚动，返回时自动继续滚动
- 滚动通知：开启/关闭滚动, 增加/减少速度操作都会在左上角显示通知. 

## 0.1.0 - feat

- 开启滚动：按 `PageDown` 键开始平滑自动滚动
- 关闭滚动：按 `PageUp` 键停止自动滚动
- 速度可配置：可以在脚本猫配置菜单中自定义每秒滚动的高度，适配不同阅读速度
