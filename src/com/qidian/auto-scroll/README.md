# 自动滚动 说明文档

## 概述

起点中文网章节页面的自动滚动脚本，通过快捷键控制页面平滑滚动，解放双手享受阅读。

## 功能特性

- 自动滚动：按 `PageDown` 键开始平滑自动滚动
- 停止滚动：按 `PageUp` 键停止自动滚动
- 速度可配置：可自定义每秒滚动的高度，适配不同阅读速度

## 使用方法

1. 安装浏览器扩展 [脚本猫](https://scriptcat.org/zh-CN)
2. 点击脚本详情页面的 **安装脚本** 按钮
3. 打开起点中文网任意章节页面
4. 按 `PageDown` 开始自动滚动，按 `PageUp` 停止

## 配置项

| 配置项         | 说明                | 默认值 |
|-------------|-------------------|-------|
| 滚动距离 (px/s) | 每秒滚动的px值，数值越大滚动越快 | 100 |

可在脚本猫扩展的脚本配置页面修改。
