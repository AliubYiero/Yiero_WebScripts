# 自动滚动 说明文档

## 概述

小说阅读页面的自动滚动脚本，通过快捷键控制页面平滑滚动，解放双手享受阅读。

> 本脚本目前仅支持能够无限滚动的阅读平台服务, 不支持自动翻页. (在后续更新计划中)
> 
> ---
> 
> 想添加支持的站点见 [问题反馈](#问题反馈) 提交功能增加请求, 或者自己新增站点 (见 [自定义网站](#自定义网站)), 但是自己新增的站点脚本更新后会被重置. 

**支持网站**

- [起点](https://www.qidian.com/): 仅支持新版能够支持无限滚动的主站, 不支持 vipreader.qidian.com
- [阅读](https://github.com/gedoor/legado): 阅读的 Web 服务 (见 [网页版看书（自带Web服务功能）](https://www.yuque.com/legado/wiki/zbpv12))

## 功能特性

- 切换滚动：按 `Space` 键开启/关闭平滑自动滚动
- 临时暂停：长按 `Space` 键临时暂停滚动，松开后自动继续滚动
- 增加速度：按 `Shift + PageUp` 键增加滚动速度
- 降低速度：按 `Shift + PageDown` 键降低滚动速度
- 速度可配置：可自定义每秒滚动的高度，适配不同阅读速度
- 智能暂停：切换标签页时自动暂停滚动，返回时自动继续滚动
- 专注模式：开启后仅在页面聚焦时滚动，失焦时停止（包括页面切换/切换到其它软件）
- 滚动通知：开启/关闭滚动，增加/减少速度操作都会在左上角显示通知

## 使用方法

1. 安装浏览器扩展 [脚本猫](https://scriptcat.org/zh-CN)
2. 点击脚本详情页面的 **安装脚本** 按钮
3. 打开起点中文网任意章节页面
4. 使用快捷键控制滚动：
   - `Space` - 开启/关闭自动滚动
   - 长按 `Space` - 临时暂停滚动
   - `Shift + PageUp` - 增加滚动速度
   - `Shift + PageDown` - 降低滚动速度

## 配置项

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 滚动距离 (px/s) | 每秒滚动的px值，数值越大滚动越快 | 100 |
| 专注模式 | 开启后仅在页面聚焦时滚动，失焦时停止（包括页面切换/切换到其它软件）；关闭时仅页面不可见时停止滚动 | 关闭 |

可在脚本猫扩展的脚本配置页面修改。

## 自定义网站

进入脚本的代码页, 最前面的几行代码中, 找到 `@match` 的一系列行: 

```js
// @match          https://www.qidian.com/chapter/*
// @match          http://192.168.5.136:1122/*
// @match          http://192.168.5.137:1122/*
// @match          http://192.168.5.138:1122/*
```

在这些行后面复制一行, 复制想要的站点的域名, 比如要新增 [QQ阅读](https://book.qq.com/) 站点: 

- 主站点为: https://book.qq.com
- 章节页面为: https://book.qq.com/book-read/24892472/1

则新增以下行, 一定要在网址后面添加加通配符 `*`: 

```js
// @match          https://book.qq.com/*
```
