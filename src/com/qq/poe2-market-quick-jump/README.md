# 流放之路2网页市集快速跳转

## 概述

适用于流放之路2（Path of Exile 2）国服网页市集，在搜索结果页面按下空格键即可自动跳转到第一个可交易商品的卖家藏身处。支持智能状态判断、自动加载下一页、双击刷新搜索等功能，大幅提升网页市集购物效率。

匹配网站：`https://poe.game.qq.com/trade2/*`

## 功能特性

- **一键空格跳转**：按下空格键自动点击第一个可用商品的"前往藏身处"按钮，并自动将页面滚动到可视范围
- **智能状态识别**：自动判断商品状态（成功跳转 / 已预定 / 已撤销），根据不同状态自动重试或提示
- **目标繁忙自动重试**：卖家藏身处繁忙时，再次点击空格可二次尝试跳转
- **道具已售自动切换**：当前商品已被购买时，自动跳转至下一个可用藏身处
- **自动加载新数据**：当前页所有商品处理完毕后，自动加载下一页搜索结果
- **双击空格刷新搜索**：当前页无可购买的道具时，在 1 秒内双击空格自动更新搜索条件
- **锁定防误触**：成功跳转后 1 秒内锁定空格键（可配置关闭或调整锁定时间），防止重复跳转
- **实时搜索模式切换**：`Ctrl + Shift + 空格` 切换实时搜索模式的开启/关闭
- **快捷刷新搜索**：`Shift + 空格` 刷新当前搜索条件

## 配置项

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| 时间锁 | 布尔 | true | 开启时间锁后，在成功跳转到目标藏身处之后的一定时间内，防止重新跳转 |
| 时间锁的持续时间 | 数字 | 1s | 时间锁的持续时长 |

## 使用方法

安装浏览器拓展 [脚本猫](https://scriptcat.org/zh-CN/)，点击脚本详情页面的 **安装脚本** 按钮。

### 基础操作

1. 访问流放之路2国服网页市集（https://poe.game.qq.com/trade2/）
2. 输入搜索条件，点击搜索获取商品列表
3. 按下 **空格键** 快速跳转至第一个可交易商品的卖家藏身处

### 快捷按键

| 按键 | 功能 |
|------|------|
| `空格` | 快速跳转至可用藏身处 / 二次尝试跳转 / 加载新数据 |
| `空格`（1秒内双击） | 当无可跳转目标时，刷新搜索条件 |
| `Shift + 空格` | 刷新当前搜索结果 |
| `Ctrl + Shift + 空格` | 切换实时搜索模式 |
