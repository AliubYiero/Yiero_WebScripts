# 流放之路2网页市集快速跳转

## 概述

适用于流放之路2（Path of Exile 2）国服网页市集，在搜索栏中按下空格键即可自动点击第一个可跳转商品旁的"前往藏身处"按钮，快速跳转到卖家藏身处进行交易。

匹配网站：`https://poe.game.qq.com/trade2/*`

## 功能特性

- **一键跳转**：在商品搜索结果页面，按下空格键自动点击第一个可用的"前往藏身处"按钮
- **无干扰**：仅在存在可跳转按钮时触发，无可用按钮时提示"未找到可以跳转的藏身处"
- **智能防误触**：跳过已禁用（已售出/已过期）的跳转按钮

## 使用方法

1. 安装浏览器拓展 [脚本猫](https://scriptcat.org/zh-CN/)
2. 点击脚本详情页面的 **安装脚本** 按钮
3. 访问流放之路2国服网页市集（https://poe.game.qq.com/trade2/）
4. 搜索商品后，在结果列表中按下空格键即可快速跳转至第一个可交易商品的藏身处
