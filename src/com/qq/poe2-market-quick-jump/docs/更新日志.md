# 更新日志

### feat - 1.1.0

**样式修改**

- 将 "前往藏身处" 按钮放大, 方便用户点击

**功能增加**

- 添加了目标繁忙时, 二次点击跳转的功能
- 添加了目标道具已被购买时, 重新跳转新的藏身处的功能 (存在问题)
- 添加了当前页的道具全部购买完毕之后, 会自动加载新的一页的功能
- 添加了当前页没有可购买的道具时, 双击空格会自动更新搜索的功能.
- 添加了时间锁, 在成功跳转目标藏身处之后, 会有 1s 的时间锁定. 防止多次点击导致触发多次跳转. 此功能可在用户配置中关闭. 

**快捷键更新**

- 添加快捷键, `Shift+空格` 可以更新搜索
- 添加快捷键, `Ctrl+Shift+空格` 可以开关实时搜索模式

### fix - 1.0.1

- 修复了 "道具不可用" 之后, 仍然会跳转至该卖家藏身处, 导致无法跳转后续卖家藏身处的问题
