# 流放之路2 批量读取道具信息

## 概述

在腾讯流放之路2交易市集页（poe.game.qq.com/trade2/*）批量读取当前页道具的名称、词缀、价格信息，导出为 CSV 表格供数据分析使用。

目前仅支持**深渊石板**的词缀识别。

## 功能特性

- 读取当前交易搜索结果页所有道具信息
- 识别道具的名称、交易ID
- 识别道具的售价及价格单位（崇高石 / 混沌石 / 神圣石）
- 解析道具词缀（前缀 / 后缀）及数值
- 支持深渊石板全部词缀识别（共 32 种词缀）
- 一键导出 CSV 文件

## 使用方法

1. 安装网页拓展 [脚本猫](https://scriptcat.org/zh-CN)
2. 安装本脚本
3. 访问腾讯流放之路2交易市集（https://poe.game.qq.com/trade2/）
4. 在搜索结果页，点击脚本猫菜单中的 **获取当前页数据**
5. 脚本会自动读取当前页所有道具信息并下载为 CSV 文件

## CSV 输出格式

| 列名 | 说明 |
| --- | --- |
| 交易单ID | 物品在交易市集中的唯一标识 |
| 物品名称 | 道具的名称 |
| 售价 | 道具的售卖价格 |
| 单位 | 价格单位（崇高石/混沌石/神圣石） |
| 前缀1 | 第一条前缀词缀名称 |
| 前缀1数值 | 第一条前缀词缀的数值 |
| 前缀2 | 第二条前缀词缀名称 |
| 前缀2数值 | 第二条前缀词缀的数值 |
| 后缀1 | 第一条后缀词缀名称 |
| 后缀1数值 | 第一条后缀词缀的数值 |
| 后缀2 | 第二条后缀词缀名称 |
| 后缀2数值 | 第二条后缀词缀的数值 |
| 读取时间 | 导出时的时间戳 |
